# Juego 2048

Proyecto de un clon del juego 2048, donde los jugadores deben combinar números en un tablero para llegar al número 2048.


# Tecnologias

Javascript

HTML/CSS

## Instalacion

1. Clonar el repositorio

```bash
  git clone https://github.com/PazTorr3s/Juego-2048.git
```

2. Entrar al repositorio

```bash
  cd Juego-2048
```

3. Abrir el proyecto con el editor de codigo (VSCode)
```bash
  code .
```
4. Abre el archivo index.html con el navegador
